# dev-groovy-cli-buckbuild-hello-world

## Description
A POC for buck build, which does
C, CPP, C#, Golang, Groovy, Java,
Kotlin, Python, Rust, and Scala.
Buck seems to extend the ant build tool
so by default it supports C, CPP, and
Java out-of-the-box.

## Tech stack
- groovy
- buck

## Docker stack
- praqma/native-buck

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
